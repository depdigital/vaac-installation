User Manual

Run the Installation

  1. Press Connect (Should be connected by default).
  2. Verify Runtime Status shows Serial: Connected.
  3. Press Run.
  4. Interact with sensors from Sensor Calibration tab and set working range for each sensors and press Save Calibration.
  5. Map sensors to a list of synths, and compose.
  6. Press Halt to stop playback.
System Shutdown

  1. Press Halt.
  2. Press Disconnect.
  3. Close the browser.
  4. Disconnect the main power.

Controls

Connect
Connects the sensor hub to the system.

Run
Run creates a new scene. Each Run command may generate different audio combinations.

Halt
Stops the current audio scene.

System Health Check

Connected
Sensor system online.

Disconnected
Sensor system offline.

Heartbeat Alive
Audio engine running.

Heartbeat Lost
Audio engine stopped.

Quick Check
If all the System Health data is GREEN, system is working fine.

Troubleshooting

No Sound

  1. Check Runtime Status from System Health tab.
  2. Verify Heartbeat is Alive.
  3. Press Run.

No Sensor Response

  1. Verify sensor hub is connected with USB. Verify Serial: Connected from the System Health tab.
  2. Move hand near sensors. You can see sensor values changing in TELEMETRY section from the System Health tab.
  3. Reconnect sensor hub. Sometimes USB cable fails, use a good quality one.
